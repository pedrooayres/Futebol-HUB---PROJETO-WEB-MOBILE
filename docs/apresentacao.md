# Apresentacao - Futebol HUB

## Slide 1 - Capa

**Titulo**
Futebol HUB

**Subtitulo**
Plataforma de scouting com dashboard interativo e CRUD em nuvem

**Aluno**
Pedro Henrique Mendonca Ayres

## Slide 2 - Proposta do Projeto

O Futebol HUB foi desenvolvido com a ideia de reunir, em uma unica interface, informacoes visuais sobre futebol e um sistema de acompanhamento de jogadores. A proposta foi criar uma experiencia moderna, com design profissional, interatividade e integracao com servicos reais na nuvem.

## Slide 3 - Objetivos

- Desenvolver um site em React/Next.js
- Aplicar CSS com identidade visual profissional
- Implementar interatividade no front-end
- Criar um CRUD funcional conectado a um back-end
- Consumir uma API publica adicional
- Publicar o projeto online

## Slide 4 - Requisitos Atendidos

- Site desenvolvido com React e Next.js
- Estilizacao customizada com CSS
- Interface responsiva para desktop e mobile
- CRUD completo da entidade `ScoutNotes`
- Integracao com o Back4App
- Consumo de API externa de futebol
- Projeto pronto para deploy no Vercel

## Slide 5 - Arquitetura da Solucao

**Front-end**
Next.js com App Router

**Back-end**
Back4App utilizando Parse REST API

**API externa**
Football Standings API para exibicao de tabela de campeonato

**Deploy**
Vercel

## Slide 6 - Entidade do CRUD

**Nome da entidade**
`ScoutNotes`

**Campos**
- `playerName`
- `club`
- `position`
- `rating`
- `status`
- `notes`

**Operacoes implementadas**
- Criar observacao
- Listar observacoes
- Editar observacao
- Excluir observacao

## Slide 7 - Funcionalidades Principais

- Dashboard inicial com indicadores dinamicos
- Secao com dados de futebol vindos de API externa
- Formulario de cadastro e edicao de observacoes
- Lista com registros cadastrados e botoes de editar e excluir
- Feedback visual para acoes do usuario

## Slide 8 - Destaques Visuais

- Layout com identidade esportiva
- Cards com profundidade e efeito glassmorphism
- Paleta com tons escuros e contrastes em verde e azul
- Hero section com chamadas principais e indicadores
- Estrutura pensada para causar boa impressao na apresentacao

## Slide 9 - Demonstracao do Sistema

Durante a apresentacao, podem ser mostrados os seguintes passos:

1. Exibicao da pagina inicial
2. Visualizacao da tabela carregada pela API externa
3. Criacao de uma nova observacao de jogador
4. Edicao do registro criado
5. Exclusao do registro

## Slide 10 - Conclusao

O projeto Futebol HUB atende aos requisitos propostos ao unir front-end moderno, integracao com banco em nuvem, consumo de API externa e publicacao web. Alem de cumprir o escopo pedido, a estrutura foi organizada para permitir evolucoes futuras, como filtros, autenticacao e dashboards adicionais.
