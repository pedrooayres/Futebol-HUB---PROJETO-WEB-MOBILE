# Roteiro do video - Futebol HUB

## Tempo total sugerido

Entre 3 e 4 minutos

## Roteiro falado pronto

### Abertura

Olá, meu nome é Pedro Henrique Mendonca Ayres e este é o projeto Futebol HUB. A proposta foi desenvolver um site em React e Next.js com visual profissional, interatividade, CRUD conectado a um back-end e integracao com uma API publica externa.

### Apresentacao da tela inicial

Na tela inicial, eu apresento um dashboard com identidade visual inspirada em analise esportiva. O objetivo foi criar uma interface moderna, responsiva e com cara de produto real. Logo no topo, temos indicadores dinamicos relacionados aos jogadores monitorados no sistema.

### API externa

Nesta secao, o sistema consome uma API publica de futebol para exibir a tabela de um campeonato. Isso atende ao requisito de integracao com uma API externa e complementa a proposta visual e funcional do site.

### CRUD - Criacao

Agora vou demonstrar o CRUD principal do projeto. Nesta area, eu consigo cadastrar uma observacao de scouting de um jogador, informando nome, clube, posicao, nota tecnica, status e observacoes. Vou preencher os campos e salvar.

### CRUD - Leitura

Depois do cadastro, o registro aparece imediatamente na lista ao lado. Isso mostra a integracao com o Back4App, que foi utilizado como back-end do projeto.

### CRUD - Edicao

Agora vou editar esse registro, alterando por exemplo a nota tecnica ou o status da avaliacao. Ao salvar novamente, os dados sao atualizados e a mudanca aparece na interface.

### CRUD - Exclusao

Por fim, vou excluir o registro para demonstrar a operacao de remocao. Assim, o sistema completa as quatro operacoes basicas do CRUD: criar, listar, editar e excluir.

### Encerramento

Para finalizar, o projeto foi publicado no Vercel e o codigo foi organizado em um repositorio no GitHub com README. Dessa forma, o Futebol HUB atende aos requisitos solicitados e apresenta uma base pronta para futuras evolucoes.

## Ordem pratica para gravar

1. Abrir a home
2. Mostrar os cards e indicadores
3. Mostrar a secao da API externa
4. Criar um registro no CRUD
5. Editar o registro
6. Excluir o registro
7. Mostrar rapidamente o GitHub e o link publicado no Vercel
